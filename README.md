# Django Advanced Projects

A repository containing multiple Django projects demonstrating various advanced and fundamental concepts in web development using Python, Django, and Django REST Framework.

## Projects Overview

This repository is structured into four distinct Django projects:

### 1. `college`
A college management application.
- **Features:** Student and Department management, image handling (media files), and CSV file upload capabilities.
- **Key Templates:** `show.html`, `show_stu.html`, `upload_csv.html`.

### 2. `drf1`
A Django REST Framework (DRF) application.
- **Features:** API development and model serialization.
- **Key Components:** `serializers.py` for API data formatting.

### 3. `project1`
A student management application focusing on core CRUD (Create, Read, Update, Delete) operations.
- **Features:** Adding, editing, and managing student records.
- **Key Templates:** `home.html`, `edit.html`.

### 4. `project2`
An application demonstrating basic authentication and data management.
- **Features:** User login functionality, alongside adding and editing records.
- **Key Templates:** `login.html`, `home.html`, `add.html`, `edit.html`.
